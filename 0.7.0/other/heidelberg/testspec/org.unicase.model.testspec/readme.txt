To make sure that the elements of this 'model extension' can be stored on a Unicase Server (which runs on your local machine),
you have to add the plug-ins to the 'Run Configuration' of the EmfStore (org.unicase.emfstore.emfstore):
 
Menu 'Run' -> 'Run Configurations...' -> select 'org.unicase.emfstore.emfstore' -> under the tab 'Plug-ins', check the following plug-ins:

- org.unicase.model.testspec
- org.unicase.model.testspec.edit
